# `@sunflower0305/claude-proxy` v1.0.0

First public npm release of `claude-proxy`, published as `@sunflower0305/claude-proxy`.

## Highlights

- publishable npm package with public scoped name `@sunflower0305/claude-proxy`
- executable `claude-proxy` CLI entrypoint
- programmatic `createApp()` export for Express-based embedding
- packaged TypeScript declarations in `dist/`
- bundled `README.md`, `LICENSE`, and `.env.example`
- Anthropic-compatible proxying for `qwen`, `deepseek`, `glm`, `minimax`, and `kimi`
- support for both non-streaming JSON responses and streaming SSE responses
- runtime provider switching through `GET/POST /api/provider`

## Installation

```bash
npx @sunflower0305/claude-proxy
```

or:

```bash
npm install -g @sunflower0305/claude-proxy
claude-proxy
```

## Quick Start

```bash
export PROVIDER=qwen
export QWEN_API_KEY=your-qwen-api-key

export ANTHROPIC_BASE_URL=http://localhost:8080
export ANTHROPIC_API_KEY=any-string-works

claude-proxy
```

## Notes

- requires Node.js `>=18`
- the proxy listens on `http://localhost:8080` by default
- clients should point `ANTHROPIC_BASE_URL` at the local proxy and can use any non-empty `ANTHROPIC_API_KEY`
