## Installation

```bash
npx @sunflower0305/claude-proxy
```

or:

```bash
npm install -g @sunflower0305/claude-proxy
claude-proxy
```

## Quick Start

```bash
export PROVIDER=qwen
export QWEN_API_KEY=your-qwen-api-key

export ANTHROPIC_BASE_URL=http://localhost:8080
export ANTHROPIC_API_KEY=any-string-works

claude-proxy
```

## Notes

- requires Node.js `>=18`
- the proxy listens on `http://localhost:8080` by default
- clients should point `ANTHROPIC_BASE_URL` at the local proxy and can use any non-empty `ANTHROPIC_API_KEY`

## Known Good Use Cases

- Claude Code with domestic Chinese LLM providers
- Claude Agent SDK with Anthropic-compatible upstreams
- local development setups that need a drop-in `/v1/messages` proxy
