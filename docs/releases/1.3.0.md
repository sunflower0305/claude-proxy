# `@sunflower0305/claude-proxy` v1.3.0

Minor release of `claude-proxy`, published as `@sunflower0305/claude-proxy`.

## Highlights

- MIMO is now available as the `mimo` provider
- the default MIMO upstream model is `mimo-v2.5-pro`
- MIMO configuration is documented in README and `.env.example`
- local integration tests cover MIMO provider switching and non-streaming proxying

## Upgrade Notes

Existing provider configuration keeps working. To use MIMO, set:

```dotenv
PROVIDER=mimo
MIMO_API_KEY=your-mimo-api-key
MIMO_MODEL=mimo-v2.5-pro
MIMO_ANTHROPIC_BASE_URL=https://api.xiaomimimo.com/anthropic
```

`MIMO_ANTHROPIC_BASE_URL` should point to the Anthropic-compatible base path. The proxy appends `/v1/messages` when forwarding requests.

## Installation

```bash
npx @sunflower0305/claude-proxy
```

